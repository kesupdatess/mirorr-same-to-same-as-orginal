# KES Updates - Donation Page

Simple donation page for KES Updates with UPI payment integration.

## Features
- Donation form with validation
- Static QR code display
- Screenshot upload
- Supabase backend integration

## Setup
1. Clone this repository
2. Open `index.html` in browser
3. Make sure QR image is at `assets/qr_codes/donation_qr_100rs.png`

## Tech Stack
- HTML/CSS/JavaScript
- Supabase (Backend)
